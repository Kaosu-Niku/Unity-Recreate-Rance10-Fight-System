# Unity Recreate Rance10 Fight System

這是一個 Unity 專案，專案內容在於還原 Rance 10 這款遊戲的戰鬥系統，
但我並沒有將圖片、音訊...等等的美術素材提交上此倉庫，避免可能會有侵權問題，
該倉庫主要目的用於做為個人的作品集以做相關學術用途，
重點在於遊戲系統的程式架構方面設計，而非美術呈現。


## 專案簡介

此專案重點在還原戰鬥的部分，
遊戲的初始畫面即可開始編輯戰鬥隊伍以及選擇要戰鬥的敵人來立刻戰鬥，
專案中已還原了大部分原遊戲中的主要人物的卡片可供戰鬥使用，
可戰鬥的敵人則是還原了原遊戲中的主要威脅敵人(魔人)，
專案亦有實作簡易的存檔功能，
戰勝過的敵人會有存檔紀錄，之後該敵人將會展示出(已擊敗)的文字提示。


## 技術分析

使用單例模式製作了大部分系統層面的功能，避免緊耦合的低級架構
ex:
戰鬥系統 = FightManager
攻擊交互系統 = BattleManager
卡片管理系統 = CardPoolManager
音效系統 = AudioManager

使用ScriptableObject定義並製作遊戲資源
ex:
角色 = CharacterSO
卡片 = CardSO
技能 = AttackSO
技能釋放條件 = AttackConditionSO
效果 = BuffSO
狀態 = StateSO

UI相關的交互都使用event事件通知觸發的方式觸發，同樣也是避免緊耦合，達成邏輯和畫面的乾淨分割

亦會注重封裝性，只公開get屬性，不隨意公開set屬性，會封裝function供外部調用來set屬性

有實作存檔功能，存檔的檔案格式使用主流的JSON格式。
